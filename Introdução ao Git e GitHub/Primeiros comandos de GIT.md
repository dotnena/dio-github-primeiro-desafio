# Primeiros comandos de GIT

git -a -> exibe arquivos ocultos

git config --global user.e-mail "email@email.com" -> altera o email de autor 

​								user.name Name -> altera o nome de autor

git add* - adiciona tudo que foi alterado p/ staged

git add NomedoArquivo - adiciona arquivo específico 

git commit -m "mensagem" - salva em commit os arquivos 

git init -> inicializa um repositório dentro do diretório

git status ->mostra o estado dos arquivos 

mv filename ./filenewfile/ -> muda o arquivo de repositório

git config --list -> lista todas as configurações 

git remote add origin https: -> adiciona repositório remoto 

git remote -v -> mostra os repositórios

git push origin master -> empurra para o repositório remoto 

git pull origin master -> faz download do repositório remoto 

git clone https:// -> clona repositório remoto para o ambiente de desenvolvimento
